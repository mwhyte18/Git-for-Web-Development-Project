Stretch Questions

    1. What is the difference between an inline element and a block element?
    A block element will take up the full amount of space on any page, meaning that nothing can be left or right of it. An inline element only takes up the width that you specify, meaning that you can have other elements in line with that element.

    2. What happens when an element is positioned absolutely? 
    If an element is positioned absolutely, has a fixed position value, meaning it will always be where you set it.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
    You can use absolute positioning. 

    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
    The <img> tag is inline-block by default, <div> <h1> <p> are display block by default, and <span> <a> are display inline by default.

    5. In your own words, explain the box model. What is th``e fix for the box model? 
The box model is what wraps whatever content you are working on. There are 3 layers in the box model, the padding, border and margin. The padding is the space around the content but is inside of the content. The border is a box around the padding and the margin is the space outside of your content.
